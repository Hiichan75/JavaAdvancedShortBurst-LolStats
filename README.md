Small Spring Boot project for the Java Advanced short-burst assignment.

## Technologies

- Java 17
- Spring Boot
- Spring Data JPA (Hibernate)
- H2 in-memory database
- Thymeleaf

## What the app does

- Add League of Legends matches:
  - summoner, champion, role, K/D/A, date, win/loss
- Store them in a database
- Show a table with all saved matches
- Calculate total games, wins, losses and winrate

## How to run

1. Clone the repository  
2. Open in IntelliJ  
3. Run `LolStatsApplication`  
4. Go to `http://localhost:8080/` in your browser
